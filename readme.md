PS插件 图片导导 和油猴脚本实现全网图片一键导入PS，支持当前已打开文件导入和新建文件导入。导入图片的图层名称会自动修改为网页端图片的名称，如果网页上的图片没有alt属性名称，则使用默认名称替代

# 介绍
> PS图片导导助手.js
>  一个油猴脚本，图片提取发送端
> 整个项目除了"PS图片导导助手.js"外是一个UXP插件，支持PS2022~2025

# 安装
脚本不支持与PS通信，所以需要搭配PS插件，安装需要两步
① 安装篡改猴脚本，云便签地址：https://webnote.cc/imgtops
② 安装PS插件，下载地址：私信微信公众号 爱吃馍 获取

关闭PS,拷贝下载后解压的文件夹“imgtops”到PS根目录下的 Plug-ins 文件夹里
比如我的是：D:\Adobe\Adobe Photoshop 2022\Plug-ins
然后双击运行开启PS插件权限.reg

打开PS,增效工具找到新安装的插件：图片导导 ，开启检测，然后就可以使用了，

# 使用
在浏览器中使用
Ctrl+左键单击图片：当前文件导入
Ctrl+Alt+左键单击图片：新建文件导入
