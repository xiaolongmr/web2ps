# 更新日志

## 版本 0.1.2

### 新增功能
- 🌸 **新增花瓣网图片支持**：专门处理花瓣网图片URL优化，删除 `_fwxxxwebp` 尺寸参数
- 🔄 **优化URL处理逻辑**：自定义平台规则优先于base64转换，避免误判

### 修复问题
- 🍎 **修复Mac兼容性问题**：
  - 简化事件监听，减少事件冲突
  - 添加剪贴板写入的备用方案
  - 优化Mac上的事件处理策略
- 🦊 **适配Safari浏览器**：
  - 添加Safari专用的事件处理逻辑
  - 实现剪贴板三重备用方案
  - 增加详细的调试信息

### 性能优化
- ⚡ **事件监听优化**：从4个事件减少到2个，提高响应稳定性
- 🔧 **跨平台适配**：同时支持Windows的Ctrl键和Mac的Command键
- 📋 **错误处理增强**：改进剪贴板操作的容错能力
